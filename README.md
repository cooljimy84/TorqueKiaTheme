# TorqueKiaTheme
Kia Theme for Torque, but old stye where you have to copy it in to a theme folder

This theme was clone from BMW E-Series Theme by Dave Ginsbach https://drive.google.com/drive/folders/1PFMnuesCCb3jIO0po0XsRfOhg57e97l2 under the Creative commons licence 4.0, EXCEPT THE INCLUDED FONTS (font.ttf, valueFont.ttf which are from KIAs branding website https://www.kia.com/au/discover-kia/brand-toolkit.html).

See: https://creativecommons.org/licenses/by-sa/4.0/

This theme has been tweaked to match the colour of the Kia 2019 proceed dashboard. I have also used AI (yuck i know) to upscale and clean the images a little to make them less square on my display.
I will be adding custom PIDs with gauages to match at somepoint, when i have scanned the car fully.
